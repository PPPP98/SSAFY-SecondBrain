# 2025.10.30

## 화면 설계 피드백

- 피그마 1차 점검 및 피드백
- 리마인더 기능 구체화
- 폰트 사이즈 조절
- 그래프 영역 구현 후 추후 수정 예정

## 기능 정리

- 노트 내 태그 기능 삭제
- 노트 검색 결과 -> 관련 (1 depth) 노트 하이라이팅
- 검색창 포커싱 -> 최근 작성 10개 노트 제공

## Neo4j 시각화

1. 시작하기

```js
import ForceGraph3D from "3d-force-graph";
const myGraph = new ForceGraph3D(myDomElement).graphData(myData);
```

2. 그래프 커스터마이징

- 노드 스타일링 : 크기, 색상, 라벨 제어
- 링크 스타일링 : 곡률, 화살표 방향, 흐름, 색상
- 렌더링/상호작용 제어 : 시점 위치 이동, 전체 보기, 애니메이션
- 힘 엔진 설정: 힘 엔진 종류, 차원 설정
