# 아이디어 구체화 - Second Brain

<br>

## 문제 정의

### 누구를 위한 서비스인가?

1. 기록을 잘 하고 있는 학습자
2. 기록을 잘 못하는 학습자
3. 학습자가 아닌 일반 사용자
- 3)의 경우 기록을 활용해야 하는 동기가 부족하여 기능에 대한 설득력이 미비함
- 1)의 경우 기존 활용하던 도구에서 변경해야 하는 이유를 제시해줘야함<br>

" 기록을 잘 못하고 있는 학습자를 위한 서비스"

<br>

### 어떤 문제를 해결하는가?

1. 정보 과부하에 따른 어려움
    - 과도한 정보로 인한 생산적 주의력 감소
    - 부정적 편향 강화
    - 비판적 사고 능력 저하
2. 정보 관리의 어려움
    - 정보의 접근성은 뛰어난 시대이지만, 역설적이게도 너무 많은 정보로 인해 정보를 저장/관리하는 것에 어려움 발생
3. 정보 관리의 중요성
    - 많은 정보가 나오는 시대에 필요한 정보를 정리하고 찾아 적절하게 사용하는 것의 중요성이 증가!

- 추가적인 근거 조사 필요

<br>

### 어떤 기능을 제공하는가?

1. 자동 접근
    - LLM, 유튜브, 블로그 등의 정보를 자동으로 가져와 저장
2. 저장
    - 가져온 데이터를 태깅, 카테고리화, 유사도 기반 매핑 등을 통해 의미를 도출한 후 저장
3. 반응
    - 저장한 정보들의 관계를 시각적으로 확인 가능
    - 자주 검색한 키워드/개념에 대해 망각곡선 기준 리마인드 알림, 퀴즈 생성 -> 중요 개념의 학습
    - 사용자의 행동과 관련하여 관련 개념을 제시, 추천
    - 일정 등 사용자 정보 기반 정보 추천 기능
